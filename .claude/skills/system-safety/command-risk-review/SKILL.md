# Skill: Command Risk Review

## Objetivo

Rever comando antes de execução.

## Avaliar

- comando;
- argumentos;
- diretório;
- ambiente;
- escrita/apagamento;
- rede;
- produção;
- dados;
- rollback.

## Output

```md
## Command Risk Review

Comando:
...

Risco:
[baixo / médio / alto / crítico]

Pode executar?
[sim / não / só com autorização]

Motivo:
...

Alternativa segura:
...
```
