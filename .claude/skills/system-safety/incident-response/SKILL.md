# Skill: Incident Response

## Objetivo

Ativar resposta inicial a incidente.

## Processo

1. Parar.
2. Conter.
3. Identificar ação.
4. Identificar ferramenta.
5. Identificar dados/áreas.
6. Classificar risco.
7. Registar sem conteúdo sensível.
8. Informar utilizador.
9. Aguardar decisão humana.

## Output

```md
## Incident Response

### Estado
[aberto / contido / em análise]

### O que aconteceu
...

### Risco
...

### Contenção
...

### Próximo passo humano
...
```
