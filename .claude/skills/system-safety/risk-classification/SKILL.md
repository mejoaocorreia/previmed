# Skill: Risk Classification

## Objetivo

Classificar rapidamente uma tarefa por risco.

## Entrada

- pedido;
- ficheiros/ferramentas envolvidos;
- ambiente;
- dados envolvidos;
- ação pretendida.

## Processo

1. Identificar dados.
2. Identificar ambiente.
3. Identificar ferramentas.
4. Identificar comandos.
5. Identificar produção.
6. Identificar rollback.
7. Classificar risco.

## Output

```md
## Risk Classification

Risco: [baixo / médio / alto / crítico]

### Motivo
...

### Bloqueios
...

### Condições para avançar
...

### Precisa de autorização?
[sim/não]
```
