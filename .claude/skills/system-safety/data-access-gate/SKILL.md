# Skill: Data Access Gate

## Objetivo

Aplicar o GDPR/RGPD Access Gate antes de aceder a dados pessoais ou sensíveis.

## Processo

1. Identificar tipo de dados.
2. Classificar dados.
3. Definir finalidade.
4. Avaliar necessidade.
5. Procurar alternativa minimizada.
6. Identificar ferramenta.
7. Definir escopo.
8. Pedir autorização.
9. Definir log.

## Regra

Sem autorização explícita, não aceder a dados laranja/vermelhos.
