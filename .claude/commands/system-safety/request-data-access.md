# Command: request-data-access

## Objetivo

Pedir autorização explícita para aceder a dados pessoais/sensíveis.

## Processo

1. Aplicar GDPR Access Gate.
2. Explicar dados e risco sem expor conteúdo.
3. Indicar ação/ferramenta.
4. Dar opções.
5. Aguardar resposta.

## Regra

Sem autorização, não aceder.
