# Command: log-sensitive-decision

## Objetivo

Registar decisão sobre dados pessoais/sensíveis sem guardar conteúdo sensível.

## Campos

- Data/Hora;
- pedido;
- tipo de dado;
- ação;
- ferramenta;
- risco;
- decisão;
- autorizado por;
- limites;
- resultado.

## Regra

Não incluir conteúdo real sensível.
