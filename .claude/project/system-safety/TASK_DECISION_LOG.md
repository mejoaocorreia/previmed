# Task Decision Log

Este ficheiro define o formato do log de decisões relevantes.

Não guardar conteúdo sensível.

## Template

| Data/Hora | Tarefa | Decisão | Motivo | Risco | Escopo | Autorizado por | Próximo passo |
|---|---|---|---|---|---|---|---|

## Usar para

- decisões de escopo;
- decisões de risco;
- decisões de ferramenta;
- decisões de rollback;
- decisões de não execução;
- decisões de escalamento.
