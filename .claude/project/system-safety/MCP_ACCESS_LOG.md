# MCP / Connector Access Log

Este ficheiro define o formato do log de acessos relevantes a connectors/MCPs.

Não guardar conteúdo sensível.

## Template

| Data/Hora | Pedido | Connector/MCP | Ação | Escopo | Nível | Dados envolvidos | Risco | Autorizado por | Resultado |
|---|---|---|---|---|---|---|---|---|---|

## Regras

- Registar acessos de risco médio/alto.
- Registar elevações de permissão.
- Registar uso de connectors com dados pessoais.
- Registar tentativas bloqueadas.
- Não guardar dados reais.
