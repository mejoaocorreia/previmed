# Sensitive Data Decision Log

Este ficheiro define o formato do log de decisões sobre dados pessoais/sensíveis.

Não guardar conteúdo sensível.

## Template

| Data/Hora | Pedido | Tipo de dado | Ação pedida | Ferramenta/Connector | Risco | Decisão | Autorizado por | Limites | Resultado |
|---|---|---|---|---|---|---|---|---|---|

## Nunca incluir

- nomes completos desnecessários;
- emails reais;
- NIF;
- moradas pessoais;
- dados médicos;
- apto/não apto;
- relatórios;
- anexos;
- passwords;
- tokens;
- conteúdo integral.
