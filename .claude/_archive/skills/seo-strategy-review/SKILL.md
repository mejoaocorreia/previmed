# seo-strategy-review

Usa para páginas/estrutura SEO.

Verificar:
- objetivo da página;
- intenção de pesquisa;
- keyword principal;
- keywords secundárias;
- cluster;
- concorrência;
- links internos;
- CTA;
- meta title/description;
- schema;
- risco de canibalização.

Output:
- prioridade;
- recomendação;
- riscos.
